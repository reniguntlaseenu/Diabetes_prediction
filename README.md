# 🩺 Diabetes Prediction Using KNN From Scratch

An interactive machine learning web application that predicts diabetes using a custom implementation of the K-Nearest Neighbors (KNN) algorithm. Built with Python and Streamlit and deployed on Streamlit Community Cloud.

### Features

* ✅ KNN implemented from scratch
* ✅ Supports Euclidean and Manhattan distance metrics
* ✅ Adjustable K value
* ✅ Interactive Streamlit interface
* ✅ Uses the Pima Indians Diabetes Dataset
* ✅ Deployed on Streamlit Community Cloud

### Tech Stack

* Python
* NumPy
* Pandas
* Scikit-learn
* Streamlit
